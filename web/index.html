Ejercicio Práctico Mod 3:1

Cálculo del Área de un Círculo con JavaScript

En esta actividad pondrás en práctica conceptos básicos de JavaScript y manipulación del DOM, aplicados a un ejercicio sencillo de matemáticas. Debes desarrollar una pequeña aplicación web que calcule el área de un círculo aplicando la fórmula: Area=π×r2

Ejercicio Práctico Mod 3:2

Identificación del Mayor y Menor con Ordenamiento Burbuja

Desarrollar un programa en JavaScript que lea tres valores ingresados por el usuario, determine el mayor y el menor número, y maneje el caso especial en que los tres valores sean iguales. Con este ejercicio reforzarás el uso de arrays, ciclos (for, do-while), condicionales y el método de ordenamiento burbuja.

Ejercicio Práctico Mod 3:3

Adivinar el número (1 - 10)

Juego con validación y control de intentos
Desarrollarás un mini-juego en JavaScript donde el computador elige un número aleatorio del 1 al 10 y el usuario debe adivinarlo en 3 intentos. Practicarás funciones, validación de entrada, uso de Math.random() / Math.floor(), control de intentos y prevención de números repetidos.

Ejercicio Práctico Mod 3:4

Conteo de Vocales en Palabras

Desarrollar un programa en JavaScript que, a partir de palabras ingresadas por el usuario, cuente la cantidad de vocales presentes en el conjunto total, utilizando funciones expresivas y métodos de manipulación de cadenas y arreglos.

Ejercicio Práctico Mod 4:1

Clases y Objetos

En esta actividad pondrás en práctica los conceptos básicos de la programación orientada a objetos (POO) en JavaScript. Aprenderás a definir clases con atributos y métodos, así como a crear objetos a partir de ellas. El objetivo es que puedas modelar entidades de la vida real y simular sus características y comportamientos en un programa sencillo.

Clase Alumno
- Define una clase que modele a un alumno de una institución educacional.
- Incluye atributos como nombre, edad, carrera, entre otros.
- Agrega un método que muestre ordenadamente todos sus atributos.

Clase Banda Musical
- Define una clase que represente a una banda musical.
- Incluye atributos como nombre, genero, integrantes y discos.
- Agrega un método para mostrar todos los atributos y otro para listar solo los discos publicados.
Instancia un objeto de la clase con los datos de tu banda favorita.

Clase Perro
- Define una clase que modele a un perro.
- Incluye atributos como nombre, raza, edad, etc.
- Agrega un método para mostrar todos sus atributos y otro para “ladrar”.
- Instancia un objeto de la clase con los datos de tu perro o de un perro famoso
